# sudoku_solver
Here is the python version of sudoku solver.

This is a basic program to solve a sudoku puzzle.

You jus need to give a input like this:

```bash
0 3 5 4 0 2 9 0 1
1 0 7 0 0 6 0 0 8
0 0 2 0 5 8 0 0 4
5 7 6 8 0 0 0 9 0
0 2 0 0 0 9 0 8 0
9 1 8 6 0 0 0 5 2
2 0 9 0 4 1 8 0 5
4 0 0 0 6 5 0 0 0
0 5 0 0 0 0 0 0 0
```

Where 0 is the place which is empty.
